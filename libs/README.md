# Compile-only dependencies

Place the following JARs here for local builds (not committed):

- `malilib-fabric-26.2-0.29.2.jar`
- `tweakeroo-fabric-26.2-0.29.2.jar`

Download from [Modrinth MaLiLib](https://modrinth.com/mod/malilib) and [Modrinth Tweakeroo](https://modrinth.com/mod/tweakeroo) (Minecraft 26.2).

GitHub Actions downloads these automatically during CI.
